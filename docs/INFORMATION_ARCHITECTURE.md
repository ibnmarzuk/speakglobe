# Information Architecture

SpeakGlobal follows a clean, single-view centric architecture.

- **Landing**
- **Authentication**
  - Sign In
  - Create Account
- **Dashboard** (Main view after login)
  - Profile Completion Card
  - Recent Sessions
  - Progress Tracker
- **AI Coach** (Practice Arena)
  - Scenario Selection
  - Active Session
  - Feedback/Analysis
- **Settings**
  - Profile
  - Account
  - Notifications
  - Billing
- **Resources**
  - Community
  - Roadmap
  - Help Center
