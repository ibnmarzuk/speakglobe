# User Journey

## Visitor Journey
1.  **Landing Page:** Visitor learns about SpeakGlobal's value.
2.  **Authentication:** Visitor clicks "Start Free AI Session", creates an account.
3.  **Dashboard:** Visitor lands in the dashboard.
4.  **AI Coaching:** Visitor starts the first AI conversation.
5.  **Feedback:** Visitor receives personalized coaching and session summary.

## Returning User Journey
1.  **Landing Page/Login:** User signs in.
2.  **Dashboard:** User sees recent progress and streak.
3.  **Practice:** User selects a scenario (e.g., "Mock Interview") and starts practice.
4.  **Coaching:** User receives feedback.
5.  **Tracking:** User tracks long-term improvement in the Progress Tracker.
