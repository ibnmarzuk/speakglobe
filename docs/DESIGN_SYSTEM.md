# Design System

SpeakGlobal uses a minimalist, premium design system.

## Typography
- Font: *Inter* (or preferred system font for readability)
- Scale: Major Second (1.125)

## Spacing
- Standard: 16px, 24px, 32px, 48px, 64px

## Colors
- **Deep Graphite:** `#1A1A1A`
- **Soft White:** `#FAFAFA`
- **Muted Purple:** `#6D28D9`
- **Warm Gold:** `#F59E0B`

## Components
- Buttons, Cards, Inputs, Dropdowns: Minimalist, clean, high-contrast.
- Animations: Subtle fade-ins.
- Loading states: Skeleton screens.
- Empty states: Helpful illustrations.
- Error states: Clear, actionable feedback.
